# DupZero App Icons
## Developed by Tavoo

## Required Files

Place these files here before running icon generation:

1. **app_icon.png** — Main app icon (1024×1024px, PNG)
   - Background: Dark (#07070F)
   - Icon: The DupZero hexagon logo in purple (#7C6FFF)

2. **app_icon_foreground.png** — Android adaptive icon foreground (1024×1024px)
   - Just the logo, transparent background

## How to Create Free

Option 1 — Canva (canva.com):
   - Free account
   - Create 1024×1024 design
   - Export as PNG

Option 2 — Figma (figma.com):
   - Free account
   - Design your icon
   - Export as PNG

## After Placing Files

Run: bash scripts/generate_icons.sh
