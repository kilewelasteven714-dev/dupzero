# DupZero — Historia Kamili ya Project
## Developed by Tavoo | Mzumbe University

---

## Muhtasari

DupZero ni app ya Flutter ya kuscan na kufuta duplicate files.
Imetengenezwa na **Tavoo** — student wa DIT II, Mzumbe University, Morogoro.

---

## V1.0 — App ya Kwanza

### Features za Msingi
- SHA-256 duplicate detection
- Perceptual image hashing (AI)
- Dark mode + color themes 8
- Firebase Auth (email + Google)
- Clean Architecture + BLoC
- Cross-platform: Android, iOS, Windows, macOS

### Architecture
- Presentation Layer (UI + BLoCs)
- Domain Layer (Entities + Use Cases)
- Data Layer (Repositories + Data Sources)
